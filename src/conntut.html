Database Connection!

Now you're all set up, let's start by connecting to your database.

How to get connected

So you've got your table, you've got some data, and you think you're some hot stuff now. Let's get in the zone to start querying some data! First, we'll have to connect to our data base using the PDO connection.

Connecting

Usually, we code in our connection commands at the top of the file, so that any code we do using the database connection can be executed without calling the connection in line. What we do to begin is create a database connection object. We can do this like so: 

	
	//We start by creating holder variables to help us
	//read the code. 
	//You will enter the right information as you see fit.
	$dbhost = 'yourhost.url.orlocation';
	$dbname = 'yourdatabasename';
	$dbuser = 'yourusername';
	$dbpass = 'yourdatabasepassword';
	
	//This line will create the new connection object.
	//I like to name it $pdo so I remember what it is.
	
	$pdo = new PDO('mysql:host='.$dbhost.';dbname='.$dbname.';charset=utf-8',$dbuser,$dbpass);
	
	//These next two lines are not necessary, but I
	//prefer to have these settings in place. These
	//will help prevent errors from breaking your
	//website. Well will cover these in detail later.
	$pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
	$pdo->setAttribute(PDO::ATTR_EMULATE_PREPARES, false);

With these in place, go ahead and reload your webpage! If there are no errors, this means that you did not have any syntax errors! That's great! We will go ahead and do a quick check to make sure that we have actually connected correctly. To do this, go ahead and try adding the following line of code any where below the 


	$pdo->query('SELECT * from Demotables');

Now go ahead and reload your web page. If no errors appear, you've done it! You're in your database and ready to do some super fine queries! Let's move on to the next section and learn about how to prepare you SQL text.
