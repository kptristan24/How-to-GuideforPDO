Database and Page Setup!

You'll setup a table, a page, and check your version of PHP.

Let's dive in!

Let's begin this turtorial by ensuring you have a web database table we can use for testing, a web page, and a current version of PHP.

MySQL table

Let's begin by going ahead to our chosen MySQL interface. My preference is of course the well renowned PHPMyAdmin, for its simple graphical interface, however any will work for our purpose, as long as you are able to execute MySQL commands. Feel free to use the command line interafce for that.

Go to your chosen SQL command executor and run the following code to create the table we'll be using to produce desired outputs. 

CREATE TABLE Demotable(
			id INT(6) AUTO_INCREMENT PRIMARY KEY,
			name VARCHAR(30) NOT NULL,
			location VARCHAR(30) NOT NULL,
			color VARCHAR(30) NOTNULL
			)

You should now have a table successfully created in your database! Let's go ahead and create a few entries so we have data to play around with.

Execute something similar to the following commands to get some data in there!

INSERT INTO Demotable (name, location, color) VALUES ('Marth', 'Fire Emblem', 'Blue')

INSERT INTO Demotable (name, location, color) VALUES ('Falcon', 'Zero-FX', 'NavyBlue')

INSERT INTO Demotable (name, location, color) VALUES ('Fox', 'Corneria', 'Orange')

Now you got some Sweet-ass data in the table! If everything ran without error here, you're ready for the next section!

Your PHP Test Page

A simple procedure here, just go ahead and navigate to your webspace, and create a php file, we'll name it, testpage.php. For now you can add one line and navigate to its location to make sure everything is correct. 

echo 'I am alive and working! Woohoo!';

Navigate to your new page at http://www.yourdomainspace.com/yourpath/testpage.php. If you can see the statement then we are set for the next step!

Check your PHP Version

In order for you to have access to the correct PDO library, you will need the most recent version of PHP installed. To keep this guide timeless, I won't say what version I am running, but I will say that if your version is older than 5.1, you will not have access to the PDO library. To check your PHP version, add the following code line to your testpage.php and navigate to our page again. 

echo 'Current PHP version: ' . phpversion();

Given you entered everything correctly, your PHP version should now be displayed by your first statement! Compare this to the internets most recent version of PHP, and proceed to update if possible. Once this is all done, let's get started using these PDO's!
