Making your things dymnamic to input

What makes this a powerful tool in my opinion, is that it is very easy to change you SQL alone, and still keep the rest of the code functioning as it should. Here's an example of you using user input to alter your SQL statements!

Querying off of User Input

Go ahead and take some user input, and just concatonate it into the string for your SQL! It'll be great! 

echo 'Query Based off of:
	<form action="yourpage.html" method="post">
		<div class="input-group">
		<input type="text" name="input" class="form-control" placeholder="Roy, Marth, Fox" aria-describedby="basic-addon1" required>
</div>
<button type="submit" class="btn btn-default">Submit</button>
</form>';

Injecting Post variables into your SQL Statement

Now we've got user data in our $_POST['input'], let's toss into our SQL statement with concatonation, and Query off of it! 

$sql = "SELECT * FROM Demotable WHERE name='. $_POST['input'] .';";
	
	$query = $pdo->prepare($sql);
	$result = $query->execute($sql);

Dynamically populating a Table from User input

This is my favourite use, is using the data we just queried to populate a table or list. This can be used in almost anyway imaginable, so be creative! Here I'll show how you can do this with a simple table. Note, I use my own table styling here, you can always trim it down. 

echo "<div class='container-fluid'>
<table class='table table-striped'>
	<thead>
		<tr>
			<th>Name</th>
			<th>Color</th>
			<th>Location</th>
		</tr>
	</thead>";

Then we'll add the PHP code part, to echo out our data with html included! This will generate a sweet table.

foreach($result as $row) {
    echo "<tr>";
		echo "<td>" . $row['name'] . "</td>";
		echo "<td>" . $row['color'] . "</td>";
		echo "<td>" . $row['location'] . "</td>";
		echo "</tr>";
}
echo "</tbody></table></div>";

Neato! You've just made your first PDO based dynamic table! Show this to your mom, she will be impressed. As far as what I have time to show you, this is about it. Play around, be creative, you now have everything you need to succeed when using PDO's, and remember, don't get frustrated when it doesn't work, you'll figure it out! Have a great day.
